# <a name="compiler-error-cs1019"></a><span data-ttu-id="677f7-103">コンパイラ エラー CS1019</span><span class="sxs-lookup"><span data-stu-id="677f7-103">Compiler Error CS1019</span></span>
<span data-ttu-id="677f7-104">オーバーロード可能な単項演算子が必要です</span><span class="sxs-lookup"><span data-stu-id="677f7-104">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="677f7-105">単項演算子のオーバーロードのような宣言が記述されていますが、演算子が不足しているか、シグネチャの不適切な場所に出現しています。</span><span class="sxs-lookup"><span data-stu-id="677f7-105">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="677f7-106">*単項演算子*は、単一のオペランドを操作する演算子です。</span><span class="sxs-lookup"><span data-stu-id="677f7-106">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="677f7-107">たとえば、`++` は単項演算子です。</span><span class="sxs-lookup"><span data-stu-id="677f7-107">For example, `++` is a unary operator.</span></span> <span data-ttu-id="677f7-108">一部の単項演算子は、`operator` キーワードを使用し、演算子の対象となる型を単一のパラメーターとして指定することでオーバーロードできます。</span><span class="sxs-lookup"><span data-stu-id="677f7-108">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="677f7-109">たとえば、`Temperature` というユーザー定義クラスで `++` 演算子をオーバーロードして `Temperature++` と記述できるようにするには、次のように演算子を定義します。</span><span class="sxs-lookup"><span data-stu-id="677f7-109">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="677f7-110">このエラーを受け取った場合は、単項演算子のオーバーロードのような宣言が記述されていますが、演算子自体が不足しているか、シグネチャの不適切な場所に出現している点が異なります。</span><span class="sxs-lookup"><span data-stu-id="677f7-110">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="677f7-111">前の例のシグネチャから `++` を削除すると、CS1019 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="677f7-111">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="677f7-112">次のコードでは CS1019 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="677f7-112">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;
   }  
  
   public static void Main()  
   {  
   }  
}  
```
