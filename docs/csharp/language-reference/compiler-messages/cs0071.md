## <a name="example"></a><span data-ttu-id="0e3af-107">例</span><span class="sxs-lookup"><span data-stu-id="0e3af-107">Example</span></span>

 <span data-ttu-id="0e3af-108">次の例では CS0071 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0e3af-108">The following sample generates CS0071.</span></span>

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
