 <span data-ttu-id="b2770-105">[switch ステートメント](../keywords/switch.md)に複数の switch セクションが含まれるとき、次のいずれかのキーワードを使用し、最後のセクションを含む、各セクションを明示的に終了する必要があります。</span><span class="sxs-lookup"><span data-stu-id="b2770-105">When a [switch statement](../keywords/switch.md) contains more than one switch section, you must explicitly terminate each section, including the last one, by using one of the following keywords:</span></span>

- [<span data-ttu-id="b2770-107">goto</span><span class="sxs-lookup"><span data-stu-id="b2770-107">goto</span></span>](../keywords/goto.md)

 <span data-ttu-id="b2770-113">次の例では CS0163 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b2770-113">The following sample generates CS0163.</span></span>  
  
```csharp  
// CS0163.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int i = 0;  
  
        switch (i)   // CS0163  
        {  
            // Compiler error CS0163 is reported on the following line.  
            case 1:  
                i++;  
            // To resolve the error, uncomment one of the following example statements.  
            // return;  
            // break;  
            // goto case 3;  
  
            case 2:  
                i++;  
                return;  
  
            case 3:  
                i = 0;  
                return;  
  
            // Compiler error CS0163 is reported on the following line.  
            default:  
                Console.WriteLine("Default");  
                // To resolve the error, uncomment the following line:  
            //break;  
        }  
    }  
```
