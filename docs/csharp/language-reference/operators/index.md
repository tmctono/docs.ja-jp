## <a name="operator-associativity"></a><span data-ttu-id="b66f1-146">演算子の結合規則</span><span class="sxs-lookup"><span data-stu-id="b66f1-146">Operator associativity</span></span>

<span data-ttu-id="b66f1-147">演算子の優先順位が同じ場合は、演算子の結合規則によって、操作の実行順序が決まります。</span><span class="sxs-lookup"><span data-stu-id="b66f1-147">When operators have the same precedence, associativity of the operators determines the order in which the operations are performed:</span></span>

- <span data-ttu-id="b66f1-148">*結合規則が左から右*の演算子は、左から右に順番に評価されます。</span><span class="sxs-lookup"><span data-stu-id="b66f1-148">*Left-associative* operators are evaluated in order from left to right.</span></span> <span data-ttu-id="b66f1-149">[代入演算子](assignment-operator.md)と [null 合体演算子](null-coalescing-operator.md)を除き、2 項演算子はすべて左からの結合です。</span><span class="sxs-lookup"><span data-stu-id="b66f1-149">Except for the [assignment operators](assignment-operator.md) and the [null-coalescing operators](null-coalescing-operator.md), all binary operators are left-associative.</span></span> <span data-ttu-id="b66f1-150">たとえば、`a + b - c` は `(a + b) - c` と評価されます。</span><span class="sxs-lookup"><span data-stu-id="b66f1-150">For example, `a + b - c` is evaluated as `(a + b) - c`.</span></span>
- <span data-ttu-id="b66f1-151">*結合規則が右から左*の演算子は、右から左に評価されます。</span><span class="sxs-lookup"><span data-stu-id="b66f1-151">*Right-associative* operators are evaluated in order from right to left.</span></span> <span data-ttu-id="b66f1-152">代入演算子、null 合体演算子、および[条件演算子`?:`](conditional-operator.md)は、右からの結合です。</span><span class="sxs-lookup"><span data-stu-id="b66f1-152">The assignment operators, the null-coalescing operators, and the [conditional operator `?:`](conditional-operator.md) are right-associative.</span></span> <span data-ttu-id="b66f1-153">たとえば、`x = y = z` は `x = (y = z)` と評価されます。</span><span class="sxs-lookup"><span data-stu-id="b66f1-153">For example, `x = y = z` is evaluated as `x = (y = z)`.</span></span>

<span data-ttu-id="b66f1-154">演算子の結合規則によって定められた評価の順序を変更するには、かっこを使用します。</span><span class="sxs-lookup"><span data-stu-id="b66f1-154">Use parentheses to change the order of evaluation imposed by operator associativity:</span></span>

```csharp-interactive
int a = 13 / 5 / 2;
int b = 13 / (5 / 2);
Console.WriteLine($"a = {a}, b = {b}");  // output: a = 1, b = 6
```
