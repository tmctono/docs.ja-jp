# <a name="compiler-error-cs0160"></a><span data-ttu-id="ac871-102">コンパイラ エラー CS0160</span><span class="sxs-lookup"><span data-stu-id="ac871-102">Compiler Error CS0160</span></span>
<span data-ttu-id="ac871-103">前の catch 句は、これまたはスーパー型 ('type') の例外を、すべて既にキャッチしています</span><span class="sxs-lookup"><span data-stu-id="ac871-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="ac871-104">一連の [catch](../language-reference/keywords/try-catch.md) ステートメントは、派生の降順になっている必要があります。</span><span class="sxs-lookup"><span data-stu-id="ac871-104">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="ac871-105">たとえば、最派生オブジェクトが最初に現れる必要があります。</span><span class="sxs-lookup"><span data-stu-id="ac871-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="ac871-106">詳細については、「[例外と例外処理](../programming-guide/exceptions/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="ac871-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="ac871-107">次の例では CS0160 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ac871-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
