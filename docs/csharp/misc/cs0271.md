# <a name="compiler-error-cs0271"></a><span data-ttu-id="de48c-102">コンパイラ エラー CS0271</span><span class="sxs-lookup"><span data-stu-id="de48c-102">Compiler Error CS0271</span></span>
<span data-ttu-id="de48c-103">get アクセサーにアクセスできないため、プロパティまたはインデクサー 'property/indexer' はこのコンテキストでは使用できません。</span><span class="sxs-lookup"><span data-stu-id="de48c-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="de48c-104">このエラーは、アクセス不能な `get` アクセサーにアクセスしようとした場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="de48c-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="de48c-105">このエラーを解決するには、アクセサーのアクセシビリティを増やすか、呼び出し元の場所を変更します。</span><span class="sxs-lookup"><span data-stu-id="de48c-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="de48c-106">詳細については、「[アクセサーのアクセシビリティ](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md)と[プロパティ](../programming-guide/classes-and-structs/properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="de48c-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="de48c-107">次の例では、CS0271 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="de48c-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
