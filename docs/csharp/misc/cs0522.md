# <a name="compiler-error-cs0522"></a><span data-ttu-id="f4e85-103">コンパイラ エラー CS0522</span><span class="sxs-lookup"><span data-stu-id="f4e85-103">Compiler Error CS0522</span></span>

<span data-ttu-id="f4e85-104">'constructor': 構造体は基底クラスのコンストラクターを呼び出すことができません</span><span class="sxs-lookup"><span data-stu-id="f4e85-104">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="f4e85-105">[構造体](../language-reference/builtin-types/struct.md) は基底クラスのコンストラクターを呼び出すことはできません。基底クラスのコンストラクターの呼び出しを削除します。</span><span class="sxs-lookup"><span data-stu-id="f4e85-105">A [struct](../language-reference/builtin-types/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="f4e85-106">次の例では CS0522 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f4e85-106">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
