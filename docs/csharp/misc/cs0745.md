# <a name="compiler-error-cs0745"></a><span data-ttu-id="d706a-102">コンパイラ エラー CS0745</span><span class="sxs-lookup"><span data-stu-id="d706a-102">Compiler Error CS0745</span></span>
<span data-ttu-id="d706a-103">コンテキスト キーワード 'by' が必要です。</span><span class="sxs-lookup"><span data-stu-id="d706a-103">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="d706a-104">`group` 句のパターンは、次の例に示すように、 `group...by` の後に続けて省略可能な `into`が続きます。</span><span class="sxs-lookup"><span data-stu-id="d706a-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

var query = from name in names  
            group name by name[0];  
```

             //...additional query clauses  
```

## <a name="example"></a><span data-ttu-id="d706a-108">例</span><span class="sxs-lookup"><span data-stu-id="d706a-108">Example</span></span>  
 <span data-ttu-id="d706a-109">次のコードでは CS0745 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d706a-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
