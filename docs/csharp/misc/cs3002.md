<span data-ttu-id="d6218-103">'method' の戻り値の型は CLS に準拠していません</span><span class="sxs-lookup"><span data-stu-id="d6218-103">Return type of 'method' is not CLS-compliant</span></span>  
  
 <span data-ttu-id="d6218-104">[public](../language-reference/keywords/public.md)、 [protected](../language-reference/keywords/protected.md)、 or `protected internal` メソッドは共通言語仕様 (CLS) に準拠した型を持つ値を返す必要があります。</span><span class="sxs-lookup"><span data-stu-id="d6218-104">A [public](../language-reference/keywords/public.md), [protected](../language-reference/keywords/protected.md), or `protected internal` method must return a value whose type is compliant with the Common Language Specification (CLS).</span></span> <span data-ttu-id="d6218-105">CLS 準拠の詳細については、「言語への[非依存性と言語に依存しないコンポーネント](../../standard/language-independence.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="d6218-105">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="d6218-106">例</span><span class="sxs-lookup"><span data-stu-id="d6218-106">Example</span></span>  
 <span data-ttu-id="d6218-107">次の例では CS3002 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d6218-107">The following example generates CS3002:</span></span>  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
