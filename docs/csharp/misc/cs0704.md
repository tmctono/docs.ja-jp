## <a name="example"></a><span data-ttu-id="9011c-107">例</span><span class="sxs-lookup"><span data-stu-id="9011c-107">Example</span></span>  

 <span data-ttu-id="9011c-108">次の例では CS0704 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="9011c-108">The following sample generates CS0704:</span></span>  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
