# <a name="compiler-error-cs0551"></a><span data-ttu-id="4d220-103">コンパイラ エラー CS0551</span><span class="sxs-lookup"><span data-stu-id="4d220-103">Compiler Error CS0551</span></span>

<span data-ttu-id="4d220-104">明示的なインターフェイス実装 'implementation' のアクセサー 'accessor' がありません</span><span class="sxs-lookup"><span data-stu-id="4d220-104">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="4d220-105">インターフェイスのプロパティを明示的に実装するクラスは、インターフェイスが定義するすべてのアクセサーを実装する必要があります。</span><span class="sxs-lookup"><span data-stu-id="4d220-105">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="4d220-106">詳細については、「 [Using Properties](../programming-guide/classes-and-structs/using-properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4d220-106">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="4d220-107">例</span><span class="sxs-lookup"><span data-stu-id="4d220-107">Example</span></span>  

 <span data-ttu-id="4d220-108">次の例では CS0551 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4d220-108">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
