## <a name="example"></a><span data-ttu-id="6adfb-109">例</span><span class="sxs-lookup"><span data-stu-id="6adfb-109">Example</span></span>  

 <span data-ttu-id="6adfb-110">次のコードでは CS0314 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6adfb-110">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
