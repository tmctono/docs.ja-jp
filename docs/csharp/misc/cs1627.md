# <a name="compiler-error-cs1627"></a><span data-ttu-id="f93d2-103">コンパイラ エラー CS1627</span><span class="sxs-lookup"><span data-stu-id="f93d2-103">Compiler Error CS1627</span></span>

<span data-ttu-id="f93d2-104">yield の戻り値の後に式が必要です。</span><span class="sxs-lookup"><span data-stu-id="f93d2-104">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="f93d2-105">式なしで `yield` が使用される場合にこのエラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="f93d2-105">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="f93d2-106">このエラーを回避するには、ステートメントに適切な式を挿入します。</span><span class="sxs-lookup"><span data-stu-id="f93d2-106">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="f93d2-107">次の例では CS1627 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f93d2-107">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
