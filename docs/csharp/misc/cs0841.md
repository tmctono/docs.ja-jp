# <a name="compiler-error-cs0841"></a><span data-ttu-id="2ce1b-103">コンパイラ エラー CS0841</span><span class="sxs-lookup"><span data-stu-id="2ce1b-103">Compiler Error CS0841</span></span>

## <a name="example"></a><span data-ttu-id="2ce1b-106">例</span><span class="sxs-lookup"><span data-stu-id="2ce1b-106">Example</span></span>

```csharp
// cs0841.cs
using System;

public class Program
{
    public static void Main()
    {
        j = 5; // CS0841
        int j;
    }
}
```

## <a name="to-correct-this-error"></a><span data-ttu-id="2ce1b-108">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="2ce1b-108">To correct this error</span></span>

```csharp

{
    public static void Main()
    {
        int j;
        j = 5;
    }
}
```
