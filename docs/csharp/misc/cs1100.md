 <span data-ttu-id="61398-105">`this` 修飾子は、メソッドが拡張メソッドであることをコンパイラに通知する、メソッドの最初のパラメーターに対してのみ指定できます。</span><span class="sxs-lookup"><span data-stu-id="61398-105">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>
