# <a name="compiler-error-cs0664"></a><span data-ttu-id="fc8c7-102">コンパイラ エラー CS0664</span><span class="sxs-lookup"><span data-stu-id="fc8c7-102">Compiler Error CS0664</span></span>
<span data-ttu-id="fc8c7-103">型 double のリテラルを暗黙的に型 'type' に変換することはできません。この型のリテラルを作成するには、'suffix' サフィックスを使用します</span><span class="sxs-lookup"><span data-stu-id="fc8c7-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="fc8c7-104">割り当てを完了できませんでした。命令を修正するには、サフィックスを使用します。</span><span class="sxs-lookup"><span data-stu-id="fc8c7-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="fc8c7-105">各型のドキュメントは、その型に対応するサフィックスを指定します。</span><span class="sxs-lookup"><span data-stu-id="fc8c7-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="fc8c7-106">変換の詳細については、「[キャストと型変換](../programming-guide/types/casting-and-type-conversions.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="fc8c7-106">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="fc8c7-107">次の例では CS0664 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="fc8c7-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
