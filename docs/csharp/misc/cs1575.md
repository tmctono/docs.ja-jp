# <a name="compiler-error-cs1575"></a><span data-ttu-id="986bc-102">コンパイラ エラー CS1575</span><span class="sxs-lookup"><span data-stu-id="986bc-102">Compiler Error CS1575</span></span>
<span data-ttu-id="986bc-103">stackalloc の式は型の後に [] が必要です</span><span class="sxs-lookup"><span data-stu-id="986bc-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="986bc-104">[stackalloc](../../csharp/language-reference/keywords/stackalloc.md)を使用して要求する割り当てのサイズは、角かっこで囲んで指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="986bc-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="986bc-105">次の例では CS1575 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="986bc-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
