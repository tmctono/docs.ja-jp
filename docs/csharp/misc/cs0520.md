# <a name="compiler-error-cs0520"></a>コンパイラ エラー CS0520
定義済みの型 'name' の宣言が正しくありません  
  
 コンパイラに必要なファイルが見つかりません。 .NET Framework 共通言語ランタイムを再インストールしてください。
