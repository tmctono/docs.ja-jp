# <a name="compiler-error-cs0424"></a><span data-ttu-id="ca9fc-103">コンパイラ エラー CS0424</span><span class="sxs-lookup"><span data-stu-id="ca9fc-103">Compiler Error CS0424</span></span>

<span data-ttu-id="ca9fc-104">'class': ComImport 属性を含むクラスは、基底クラスを指定できません。</span><span class="sxs-lookup"><span data-stu-id="ca9fc-104">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="ca9fc-105"><xref:System.Runtime.InteropServices.ComImportAttribute> 属性を指定することは、クラスの実装が COM モジュールからインポートされることを意味します。</span><span class="sxs-lookup"><span data-stu-id="ca9fc-105">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="ca9fc-106">追加のメソッドや、基底クラスから継承されたフィールドは COM モジュールで定義されている実装に追加することができません。</span><span class="sxs-lookup"><span data-stu-id="ca9fc-106">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>  
  
 <span data-ttu-id="ca9fc-107">次の例では CS0424 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ca9fc-107">The following sample generates CS0424:</span></span>

