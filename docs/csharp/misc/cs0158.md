 <span data-ttu-id="400da-107">次の例では CS0158 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="400da-107">The following sample generates CS0158:</span></span>  
  
```csharp

   {  
      public static void Main()

         }  
      }  
   }  
}
```
