# <a name="compiler-error-cs1597"></a>コンパイラ エラー CS1597

メソッドまたはアクセサー ブロックの後のセミコロンの使用が正しくありません。  
  
 メソッドまたはアクセサー ブロックの終了に、セミコロンは不要です (または許可されていません)。  
  
 次の例では CS1597 が生成されます。  
  
```csharp  
// CS1597.cs  
class TestClass  
{  
   public static void Main()  
   {  
   };   // CS1597, remove semicolon  
}  
```
