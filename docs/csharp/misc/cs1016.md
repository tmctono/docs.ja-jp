<span data-ttu-id="a4bde-104">名前付き属性引数が必要です。</span><span class="sxs-lookup"><span data-stu-id="a4bde-104">Named attribute argument expected</span></span>  
  
 <span data-ttu-id="a4bde-105">名前のない属性引数は、名前付き引数の前に使用する必要があります。</span><span class="sxs-lookup"><span data-stu-id="a4bde-105">Unnamed attribute arguments must appear before the named arguments.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a4bde-106">例</span><span class="sxs-lookup"><span data-stu-id="a4bde-106">Example</span></span>  

 <span data-ttu-id="a4bde-107">次の例では CS1016 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a4bde-107">The following sample generates CS1016:</span></span>  
  
```csharp  
// CS1016.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class HelpAttribute : Attribute  
{  
    public HelpAttribute(string url)   // url is a positional parameter  
    {  
        m_url = url;  
    }  
  
    public string Topic = null;      // Topic is a named parameter  
    private string m_url = null;  
}  
  
[HelpAttribute(Topic="Samples", "http://intranet/inhouse")]   // CS1016  
// try the following line instead  
//[HelpAttribute("http://intranet/inhouse", Topic="Samples")]  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```
