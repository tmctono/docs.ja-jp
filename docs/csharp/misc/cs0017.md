# <a name="compiler-error-cs0017"></a><span data-ttu-id="6bdd4-102">コンパイラ エラー CS0017</span><span class="sxs-lookup"><span data-stu-id="6bdd4-102">Compiler Error CS0017</span></span>
<span data-ttu-id="6bdd4-103">プログラム 'output file name' に、複数のエントリ ポイントが定義されています。</span><span class="sxs-lookup"><span data-stu-id="6bdd4-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="6bdd4-104">エントリ ポイントを含む型を指定するために、/main を使用してコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="6bdd4-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="6bdd4-105">プログラムには、 [Main](../programming-guide/main-and-command-args/index.md) メソッドを 1 つのみ指定できます。</span><span class="sxs-lookup"><span data-stu-id="6bdd4-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="6bdd4-106">このエラーを解決するには、コード内のすべての Main メソッドを削除するか、または[-main](../language-reference/compiler-options/main-compiler-option.md)コンパイラオプションを使用して、使用する main メソッドを指定します。</span><span class="sxs-lookup"><span data-stu-id="6bdd4-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="6bdd4-107">次の例では CS0017 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6bdd4-107">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
