# <a name="compiler-error-cs0506"></a><span data-ttu-id="52b77-103">コンパイラ エラー CS0506</span><span class="sxs-lookup"><span data-stu-id="52b77-103">Compiler Error CS0506</span></span>

<span data-ttu-id="52b77-104">'function1': 継承されたメンバー 'function2' は "virtual"、"abstract"、または "override" に設定されていないためオーバーライドできません</span><span class="sxs-lookup"><span data-stu-id="52b77-104">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="52b77-105">**virtual**、 **abstract**、または `override`として明示的にマークされていないメソッドがオーバーライドされました。</span><span class="sxs-lookup"><span data-stu-id="52b77-105">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="52b77-106">次の例では CS0506 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="52b77-106">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
