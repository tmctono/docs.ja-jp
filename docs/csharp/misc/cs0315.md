<span data-ttu-id="e4690-103">型 ' valueType ' は、ジェネリック型またはメソッド ' typeormethod\<T > ' で型パラメーター ' t ' として使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="e4690-103">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="e4690-104">'valueType' から 'referenceType' へのボックス変換がありません。</span><span class="sxs-lookup"><span data-stu-id="e4690-104">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>

## <a name="example"></a><span data-ttu-id="e4690-108">例</span><span class="sxs-lookup"><span data-stu-id="e4690-108">Example</span></span>  
 <span data-ttu-id="e4690-109">次の例では、CS0315 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e4690-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
