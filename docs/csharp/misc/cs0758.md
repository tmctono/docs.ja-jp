# <a name="compiler-error-cs0758"></a><span data-ttu-id="68a85-102">コンパイラ エラー CS0758</span><span class="sxs-lookup"><span data-stu-id="68a85-102">Compiler Error CS0758</span></span>
<span data-ttu-id="68a85-103">部分メソッド宣言は、両方とも params パラメーターを使用するか、両方とも params パラメーターを使用しないかのいずれかである必要があります。</span><span class="sxs-lookup"><span data-stu-id="68a85-103">Both partial method declarations must use a params parameter or neither may use a params parameter</span></span>  
  
 <span data-ttu-id="68a85-104">部分メソッドの一部が `params` パラメーターを指定する場合、その他の部分もこのパラメーターを指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="68a85-104">If one part of a partial method specifies a `params` parameter, the other part must specify one also.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="68a85-105">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="68a85-105">To correct this error</span></span>  
  
1. <span data-ttu-id="68a85-106">`params` 修飾子をメソッドの一部に追加するか、他の部分からこの修飾子を削除します。</span><span class="sxs-lookup"><span data-stu-id="68a85-106">Either add the `params` modifier in one part of the method, or remove it in the other.</span></span>  
  
## <a name="example"></a><span data-ttu-id="68a85-107">例</span><span class="sxs-lookup"><span data-stu-id="68a85-107">Example</span></span>

            return 1;  
        }  
  
    }
