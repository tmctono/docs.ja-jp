# <a name="compiler-error-cs0466"></a><span data-ttu-id="9c394-103">コンパイラ エラー CS0466</span><span class="sxs-lookup"><span data-stu-id="9c394-103">Compiler Error CS0466</span></span>

<span data-ttu-id="9c394-104">'method1' には params パラメーターがないため、'method2' は params パラメーターを持つことができません</span><span class="sxs-lookup"><span data-stu-id="9c394-104">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="9c394-105">`params` パラメーターは、実装インターフェイスで使用していない場合、クラス メンバーで使用できません。</span><span class="sxs-lookup"><span data-stu-id="9c394-105">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9c394-106">例</span><span class="sxs-lookup"><span data-stu-id="9c394-106">Example</span></span>  

 <span data-ttu-id="9c394-107">次の例では CS0466 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="9c394-107">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
