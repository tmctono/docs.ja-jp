# <a name="compiler-error-cs0415"></a><span data-ttu-id="3ff18-102">コンパイラ エラー CS0415</span><span class="sxs-lookup"><span data-stu-id="3ff18-102">Compiler Error CS0415</span></span>
<span data-ttu-id="3ff18-103">'IndexerName' 属性は、明示的なインターフェイス メンバー宣言ではないインデクサー上でのみ有効です。</span><span class="sxs-lookup"><span data-stu-id="3ff18-103">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="3ff18-104">このエラーは、インターフェイスの明示的な実装であるインデクサーで IndexerName 属性を使用する場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="3ff18-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="3ff18-105">このエラーを回避するには、可能であれば、インデクサーの宣言からインターフェイスの名前を削除します。</span><span class="sxs-lookup"><span data-stu-id="3ff18-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="3ff18-106">詳細については、「 [IndexerNameAttribute クラス](xref:System.Runtime.CompilerServices.IndexerNameAttribute)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="3ff18-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="3ff18-107">次の例では CS0415 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3ff18-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
