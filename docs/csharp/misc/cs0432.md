# <a name="compiler-error-cs0432"></a><span data-ttu-id="b6b0f-103">コンパイラ エラー CS0432</span><span class="sxs-lookup"><span data-stu-id="b6b0f-103">Compiler Error CS0432</span></span>
<span data-ttu-id="b6b0f-104">エイリアス 'identifier' が見つかりません</span><span class="sxs-lookup"><span data-stu-id="b6b0f-104">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="b6b0f-105">このエラーは、エイリアスではない識別子の右側に "::" を使用した場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="b6b0f-105">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="b6b0f-106">エラーを解決するには、代わりに "." を使用します。</span><span class="sxs-lookup"><span data-stu-id="b6b0f-106">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="b6b0f-107">次の例では、CS0432 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b6b0f-107">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
