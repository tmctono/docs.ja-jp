# <a name="compiler-error-cs0629"></a><span data-ttu-id="e958f-102">コンパイラ エラー CS0629</span><span class="sxs-lookup"><span data-stu-id="e958f-102">Compiler Error CS0629</span></span>
<span data-ttu-id="e958f-103">条件付きのメンバー 'member' は型 'Type Name' にインターフェイス メンバー 'base class member' を実装することはできません</span><span class="sxs-lookup"><span data-stu-id="e958f-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="e958f-104">[条件付き](../programming-guide/concepts/attributes/common-attributes.md#Conditional) 属性は、インターフェイスの実装で使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="e958f-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="e958f-105">次の例では CS0629 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e958f-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
