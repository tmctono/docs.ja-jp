<span data-ttu-id="f3a3d-103">配列が、コンパイル時に知られていないディメンションを初期化します。</span><span class="sxs-lookup"><span data-stu-id="f3a3d-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="f3a3d-104">次のコードでは、このエラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="f3a3d-104">The following code generates this error.</span></span>  
  
```vb  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="f3a3d-105">次のコードでは、エラーが回避されます。</span><span class="sxs-lookup"><span data-stu-id="f3a3d-105">The following code avoids the error.</span></span>  
  
```vb

    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```
