# <a name="specified-dll-function-not-found"></a>指定された DLL 関数が見つかりません

ユーザーのライブラリ リファレンス内にダイナミック リンク ライブラリ (DLL) が見つかりましたが、この DLL 内には指定された DLL 関数がありませんでした。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 関数の宣言で正しい序数を指定します。  
  
2. DLL の名前とエイリアスが正しいことを確認します。
