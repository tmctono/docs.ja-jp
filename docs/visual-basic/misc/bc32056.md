<span data-ttu-id="4d983-103">ジェネリック型の `Implements` 句は、実装するメンバーとして型パラメーターを指定しています。</span><span class="sxs-lookup"><span data-stu-id="4d983-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>
