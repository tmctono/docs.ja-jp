<span data-ttu-id="78c8c-103">Entity Data Model (EDM) は、概念モデルで[プロパティ](property.md)を定義するために使用される一連の抽象プリミティブデータ型 (String、Boolean、Int32 など) をサポートしています。</span><span class="sxs-lookup"><span data-stu-id="78c8c-103">The Entity Data Model (EDM) supports a set of abstract primitive data types (such as String, Boolean, Int32, and so on) that are used to define [properties](property.md) in a conceptual model.</span></span> <span data-ttu-id="78c8c-104">これらのプリミティブ データ型は、SQL Server データベースや共通言語ランタイム (CLR) などのストレージ環境またはホスト環境でサポートされる、実際のプリミティブ データ型のプロキシです。</span><span class="sxs-lookup"><span data-stu-id="78c8c-104">These primitive data types are proxies for actual primitive data types that are supported in the storage or hosting environment, such as a SQL Server database or the common language runtime (CLR).</span></span> <span data-ttu-id="78c8c-105">EDM では、プリミティブ データ型に対する演算や変換のセマンティクスを定義していません。これらのセマンティクスは、ストレージ環境またはホスト環境で定義されます。</span><span class="sxs-lookup"><span data-stu-id="78c8c-105">The EDM does not define the semantics of operations or conversions over primitive data types; these semantics are defined by the storage or hosting environment.</span></span> <span data-ttu-id="78c8c-106">通常、EDM のプリミティブ データ型は、ストレージ環境またはホスト環境の対応プリミティブ データ型にマップされます。</span><span class="sxs-lookup"><span data-stu-id="78c8c-106">Typically, primitive data types in the EDM are mapped to corresponding primitive data types in the storage or hosting environment.</span></span> <span data-ttu-id="78c8c-107">Entity Framework が EDM のプリミティブ型を SQL Server データ型にマップする方法の詳細については、「 [Entity frameworktypes の SqlClient](./ef/sqlclient-for-ef-types.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="78c8c-107">For information about how the Entity Framework maps primitive types in the EDM to SQL Server data types, see [SqlClient for Entity FrameworkTypes](./ef/sqlclient-for-ef-types.md).</span></span>  
  
> [!NOTE]
> <span data-ttu-id="78c8c-108">EDM では、プリミティブ データ型のコレクションをサポートしていません。</span><span class="sxs-lookup"><span data-stu-id="78c8c-108">The EDM does not support collections of primitive data types.</span></span>  
  
 <span data-ttu-id="78c8c-109">EDM での構造化データ型の詳細については、「[エンティティ型](entity-type.md)」と「[複合型](complex-type.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="78c8c-109">For information about structured data types in the EDM, see [entity type](entity-type.md) and [complex type](complex-type.md).</span></span>
