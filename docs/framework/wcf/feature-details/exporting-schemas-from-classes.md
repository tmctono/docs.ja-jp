 <span data-ttu-id="13087-107">は、 `XmlSchemaSet` XSD スキーマドキュメントのセットを表す .NET Framework のスキーマオブジェクトモデル (SOM) の一部です。</span><span class="sxs-lookup"><span data-stu-id="13087-107">The `XmlSchemaSet` is part of the .NET Framework’s Schema Object Model (SOM) that represents a set of XSD Schema documents.</span></span> <span data-ttu-id="13087-108">XSD ドキュメントを `XmlSchemaSet`から生成するために、 <xref:System.Xml.Schema.XmlSchemaSet.Schemas%2A> クラスの `XmlSchemaSet` プロパティから取得した、スキーマのコレクションを使います。</span><span class="sxs-lookup"><span data-stu-id="13087-108">To create XSD documents from an `XmlSchemaSet`, use the collection of schemas from the <xref:System.Xml.Schema.XmlSchemaSet.Schemas%2A> property of the `XmlSchemaSet` class.</span></span> <span data-ttu-id="13087-109">次に各 <xref:System.Xml.Schema.XmlSchema> オブジェクトを、 <xref:System.Xml.Serialization.XmlSerializer>を使ってシリアル化します。</span><span class="sxs-lookup"><span data-stu-id="13087-109">Then serialize each <xref:System.Xml.Schema.XmlSchema> object using the <xref:System.Xml.Serialization.XmlSerializer>.</span></span>  
  
#### <a name="to-export-schemas"></a><span data-ttu-id="13087-110">スキーマをエクスポートするには</span><span class="sxs-lookup"><span data-stu-id="13087-110">To export schemas</span></span>  
  
1. <span data-ttu-id="13087-111"><xref:System.Runtime.Serialization.XsdDataContractExporter> のインスタンスを作成します。</span><span class="sxs-lookup"><span data-stu-id="13087-111">Create an instance of the <xref:System.Runtime.Serialization.XsdDataContractExporter>.</span></span>

     <span data-ttu-id="13087-122">`Export` メソッドを何度も呼び出すと、同じ `XmlSchemaSet`に複数の項目が追加されます。</span><span class="sxs-lookup"><span data-stu-id="13087-122">Multiple calls to the `Export` method results in multiple items being added to the same `XmlSchemaSet`.</span></span> <span data-ttu-id="13087-123">`XmlSchemaSet` に既に存在する型は生成されません。</span><span class="sxs-lookup"><span data-stu-id="13087-123">A type is not generated into the `XmlSchemaSet` if it already exists there.</span></span> <span data-ttu-id="13087-124">したがって、 `Export` クラスのインスタンスを複数作成するのではなく、同じ `XsdDataContractExporter` に対して `XsdDataContractExporter` を複数回呼び出すようにします。</span><span class="sxs-lookup"><span data-stu-id="13087-124">Therefore, calling `Export` multiple times on the same `XsdDataContractExporter` is preferable to creating multiple instances of the `XsdDataContractExporter` class.</span></span> <span data-ttu-id="13087-125">これにより、同じスキーマ型が重複して生成される状況を回避できます。</span><span class="sxs-lookup"><span data-stu-id="13087-125">This avoids duplicate schema types from being generated.</span></span>  
  
    > [!NOTE]
    > <span data-ttu-id="13087-126">エクスポート処理中に障害が発生した場合、 `XmlSchemaSet` は予測できない状態になります。</span><span class="sxs-lookup"><span data-stu-id="13087-126">If there is a failure during export, the `XmlSchemaSet` will be in an unpredictable state.</span></span>  
  
5. <span data-ttu-id="13087-127"><xref:System.Xml.Schema.XmlSchemaSet> プロパティを介して、 <xref:System.Runtime.Serialization.XsdDataContractExporter.Schemas%2A> にアクセスします。</span><span class="sxs-lookup"><span data-stu-id="13087-127">Access the <xref:System.Xml.Schema.XmlSchemaSet> through the <xref:System.Runtime.Serialization.XsdDataContractExporter.Schemas%2A> property.</span></span>
