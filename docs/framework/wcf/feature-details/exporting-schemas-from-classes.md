<span data-ttu-id="a93e7-103">データ コントラクト モデルで使用されているクラスから XML スキーマ定義言語 (XSD) スキーマを生成するには、 <xref:System.Runtime.Serialization.XsdDataContractExporter> クラスを使います。</span><span class="sxs-lookup"><span data-stu-id="a93e7-103">To generate XML Schema definition language (XSD) schemas from classes that are used in the data contract model, use the <xref:System.Runtime.Serialization.XsdDataContractExporter> class.</span></span> <span data-ttu-id="a93e7-104">このトピックでは、スキーマの作成手順を説明します。</span><span class="sxs-lookup"><span data-stu-id="a93e7-104">This topic describes the process for creating schemas.</span></span>  
  
## <a name="the-export-process"></a><span data-ttu-id="a93e7-105">エクスポート処理</span><span class="sxs-lookup"><span data-stu-id="a93e7-105">The Export Process</span></span>  
 <span data-ttu-id="a93e7-106">スキーマのエクスポート処理では、まず型の定義をエクスポートするので、型定義を XML 形式で記述した <xref:System.Xml.Schema.XmlSchemaSet> を生成します。</span><span class="sxs-lookup"><span data-stu-id="a93e7-106">The schema export process starts with one or more types and produces an <xref:System.Xml.Schema.XmlSchemaSet> that describes the XML projection of these types.</span></span>  
  
 <span data-ttu-id="a93e7-107">は`XmlSchemaSet` 、XSD スキーマドキュメントのセットを表す .NET Framework のスキーマオブジェクトモデル (SOM) の一部です。</span><span class="sxs-lookup"><span data-stu-id="a93e7-107">The `XmlSchemaSet` is part of the .NET Framework’s Schema Object Model (SOM) that represents a set of XSD Schema documents.</span></span> <span data-ttu-id="a93e7-108">XSD ドキュメントを `XmlSchemaSet`から生成するために、 <xref:System.Xml.Schema.XmlSchemaSet.Schemas%2A> クラスの `XmlSchemaSet` プロパティから取得した、スキーマのコレクションを使います。</span><span class="sxs-lookup"><span data-stu-id="a93e7-108">To create XSD documents from an `XmlSchemaSet`, use the collection of schemas from the <xref:System.Xml.Schema.XmlSchemaSet.Schemas%2A> property of the `XmlSchemaSet` class.</span></span> <span data-ttu-id="a93e7-109">次に各 <xref:System.Xml.Schema.XmlSchema> オブジェクトを、 <xref:System.Xml.Serialization.XmlSerializer>を使ってシリアル化します。</span><span class="sxs-lookup"><span data-stu-id="a93e7-109">Then serialize each <xref:System.Xml.Schema.XmlSchema> object using the <xref:System.Xml.Serialization.XmlSerializer>.</span></span>  
  
#### <a name="to-export-schemas"></a><span data-ttu-id="a93e7-110">スキーマをエクスポートするには</span><span class="sxs-lookup"><span data-stu-id="a93e7-110">To export schemas</span></span>  
  
1. <span data-ttu-id="a93e7-111"><xref:System.Runtime.Serialization.XsdDataContractExporter>のインスタンスを作成します。</span><span class="sxs-lookup"><span data-stu-id="a93e7-111">Create an instance of the <xref:System.Runtime.Serialization.XsdDataContractExporter>.</span></span>  
  
2. <span data-ttu-id="a93e7-112">省略可能です。</span><span class="sxs-lookup"><span data-stu-id="a93e7-112">Optional.</span></span> <span data-ttu-id="a93e7-113"><xref:System.Xml.Schema.XmlSchemaSet> をコンストラクターに渡します。</span><span class="sxs-lookup"><span data-stu-id="a93e7-113">Pass an <xref:System.Xml.Schema.XmlSchemaSet> in the constructor.</span></span> <span data-ttu-id="a93e7-114">この場合、空の <xref:System.Xml.Schema.XmlSchemaSet> から開始するのではなく、スキーマのエクスポート時に生成されたスキーマが、この <xref:System.Xml.Schema.XmlSchemaSet>インスタンスに追加されます。</span><span class="sxs-lookup"><span data-stu-id="a93e7-114">In this case, the schema generated during the schema export is added to this <xref:System.Xml.Schema.XmlSchemaSet> instance instead of starting with a blank <xref:System.Xml.Schema.XmlSchemaSet>.</span></span>  
  
3. <span data-ttu-id="a93e7-115">省略可能です。</span><span class="sxs-lookup"><span data-stu-id="a93e7-115">Optional.</span></span> <span data-ttu-id="a93e7-116"><xref:System.Runtime.Serialization.XsdDataContractExporter.CanExport%2A> メソッドのいずれかを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="a93e7-116">Call one of the <xref:System.Runtime.Serialization.XsdDataContractExporter.CanExport%2A> methods.</span></span> <span data-ttu-id="a93e7-117">指定された型がエクスポート可能かどうかを判定するメソッドです。</span><span class="sxs-lookup"><span data-stu-id="a93e7-117">The method determines whether the specified type can be exported.</span></span> <span data-ttu-id="a93e7-118">このメソッドには、次に説明する `Export` メソッドと同じオーバーロードがあります。</span><span class="sxs-lookup"><span data-stu-id="a93e7-118">The method has the same overloads as the `Export` method in the next step.</span></span>  
  
4. <span data-ttu-id="a93e7-119"><xref:System.Runtime.Serialization.XsdDataContractExporter.Export%2A> メソッドのいずれかを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="a93e7-119">Call one of the <xref:System.Runtime.Serialization.XsdDataContractExporter.Export%2A> methods.</span></span> <span data-ttu-id="a93e7-120"><xref:System.Type>、 <xref:System.Collections.Generic.List%601> オブジェクトの `Type` 、または <xref:System.Collections.Generic.List%601> オブジェクトの <xref:System.Reflection.Assembly> を受け取る 3 種類のオーバーロードがあります。</span><span class="sxs-lookup"><span data-stu-id="a93e7-120">There are three overloads taking a <xref:System.Type>, a <xref:System.Collections.Generic.List%601> of `Type` objects, or a <xref:System.Collections.Generic.List%601> of <xref:System.Reflection.Assembly> objects.</span></span> <span data-ttu-id="a93e7-121">3 つ目のメソッドの場合、指定されたアセンブリのすべての型がエクスポートされます。</span><span class="sxs-lookup"><span data-stu-id="a93e7-121">In the last case, all types in all the given assemblies are exported.</span></span>  
  
     <span data-ttu-id="a93e7-122">`Export` メソッドを何度も呼び出すと、同じ `XmlSchemaSet`に複数の項目が追加されます。</span><span class="sxs-lookup"><span data-stu-id="a93e7-122">Multiple calls to the `Export` method results in multiple items being added to the same `XmlSchemaSet`.</span></span> <span data-ttu-id="a93e7-123">`XmlSchemaSet` に既に存在する型は生成されません。</span><span class="sxs-lookup"><span data-stu-id="a93e7-123">A type is not generated into the `XmlSchemaSet` if it already exists there.</span></span> <span data-ttu-id="a93e7-124">したがって、 `Export` クラスのインスタンスを複数作成するのではなく、同じ `XsdDataContractExporter` に対して `XsdDataContractExporter` を複数回呼び出すようにします。</span><span class="sxs-lookup"><span data-stu-id="a93e7-124">Therefore, calling `Export` multiple times on the same `XsdDataContractExporter` is preferable to creating multiple instances of the `XsdDataContractExporter` class.</span></span> <span data-ttu-id="a93e7-125">これにより、同じスキーマ型が重複して生成される状況を回避できます。</span><span class="sxs-lookup"><span data-stu-id="a93e7-125">This avoids duplicate schema types from being generated.</span></span>  
  
    > [!NOTE]
    > <span data-ttu-id="a93e7-126">エクスポート処理中に障害が発生した場合、 `XmlSchemaSet` は予測できない状態になります。</span><span class="sxs-lookup"><span data-stu-id="a93e7-126">If there is a failure during export, the `XmlSchemaSet` will be in an unpredictable state.</span></span>  
  
5. <span data-ttu-id="a93e7-127"><xref:System.Xml.Schema.XmlSchemaSet> プロパティを介して、 <xref:System.Runtime.Serialization.XsdDataContractExporter.Schemas%2A> にアクセスします。</span><span class="sxs-lookup"><span data-stu-id="a93e7-127">Access the <xref:System.Xml.Schema.XmlSchemaSet> through the <xref:System.Runtime.Serialization.XsdDataContractExporter.Schemas%2A> property.</span></span>
