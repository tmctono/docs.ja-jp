## <a name="the-elements-of-a-binding"></a><span data-ttu-id="af0fd-116">バインディングの要素</span><span class="sxs-lookup"><span data-stu-id="af0fd-116">The Elements of a Binding</span></span>

- <span data-ttu-id="af0fd-128"><xref:System.ServiceModel.WSHttpBinding> : WS-\* プロトコルに準拠するエンドポイントへの接続に適した相互運用可能なバインディング。</span><span class="sxs-lookup"><span data-stu-id="af0fd-128"><xref:System.ServiceModel.WSHttpBinding>: An interoperable binding suitable for connecting to endpoints that conform to the WS-\* protocols.</span></span>

- <span data-ttu-id="af0fd-131"><xref:System.ServiceModel.NetTcpBinding>: このバインディングは HTTP バインディングよりも高いパフォーマンスを提供し、ローカルネットワークでの使用に最適です。</span><span class="sxs-lookup"><span data-stu-id="af0fd-131"><xref:System.ServiceModel.NetTcpBinding>: This binding offers higher performance than HTTP bindings and is ideal for use in a local network.</span></span>
  
 <span data-ttu-id="af0fd-132">WCF に用意されているすべてのバインディングの完全な一覧と説明については、「[システム指定のバインディング](system-provided-bindings.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="af0fd-132">For a complete list, with descriptions, of all the WCF-provided bindings, see [System-Provided Bindings](system-provided-bindings.md).</span></span>
